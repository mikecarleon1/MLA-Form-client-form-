MLA Transfer form — quick fill

Paste the whole call note below, then click a button. Your browser opens the Google Form with fields filled in. Always check before you submit.

For messy or unusual notes, use the AI + few-shot path on your PC: python scripts/mla_form_llm_prefill.py yournote.txt --open (needs openai and OPENAI_API_KEY). Add real examples to scripts/mla_few_shot_examples.json so it learns your formats. To bulk-check parsing without opening the form, run python scripts/test_mla_prefill_regression.py or use Regression tests below.

Payment details in a link can sit in browser history. On shared PCs, prefer “without payment” and type card or bank on the form.

If nothing opens, your browser blocked the pop-up — use Copy link and paste into the address bar, or allow pop-ups for this file.
